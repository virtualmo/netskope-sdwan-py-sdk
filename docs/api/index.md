# API Reference

This section documents the package-supported Python API using `mkdocstrings`.

The reference is organized by module so it stays maintainable as the SDK grows:

- package exports and top-level entry points
- client configuration and tenant resolution helpers
- resource manager modules
- legacy v1 namespace
- models, enums, and exceptions

Internal helper modules are intentionally kept light in the navigation. The focus here is the supported SDK surface that callers are most likely to import or interact with directly.
